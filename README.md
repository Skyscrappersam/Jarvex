# 🤖 Jarvex

**Jarvex** is an AI-powered personal assistant built with Python, designed for desktop and future mobile platforms. It provides voice interaction, intelligent conversations, internet search, automation, memory, and productivity features.

---

# ✨ Features

- 🎤 Voice Recognition (Whisper)
- 🔊 Text-to-Speech Responses
- 💬 AI Chat
- 🌐 Internet Search
- 🌦️ Live Weather
- 📅 Current Date & Time
- 🧠 Long-term Memory
- ⚙️ Open Applications (Calculator, Notepad, etc.)
- 🖥️ Modern GUI
- ⌨️ Text & Voice Input

# 🛠️ Installation

## Clone the repository

```bash
git clone https://github.com/Skyscrappersam/Jarvex.git
```

Go into the project:

```bash
cd Jarvex
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jarvex:

```bash
python launch_gui.py
```
