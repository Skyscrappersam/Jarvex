# 🤖 Jarvex

> An AI-powered personal assistant built with Python, designed for desktop and future mobile platforms.

---

## ✨ Features

- 🎤 Voice Recognition (Faster-Whisper)
- 🔊 Text-to-Speech
- 💬 AI Chat
- 🌐 Internet Search
- 🌦️ Live Weather
- 📅 Current Date & Time
- 🧠 Memory System
- ⚙️ App Automation
- 🖥️ Modern GUI
- ⌨️ Text & Voice Input

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Skyscrappersam/Jarvex.git
```

### Open the project

```bash
cd Jarvex
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run Jarvex

```bash
python launch_gui.py
```
